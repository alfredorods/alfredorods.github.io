Descripción:

Gestionar es una App de logística y servicios de delivery para empresas que integra diferentes marketplace como Mercado libre, Woocommerce y Shopify para sincronizar ventas de diferentes proveedores y prestar un servicio de paquetería a los mismos. Todo desarrollado bajo un ambiente cloud con diferentes tecnologías tales como Node Js, Angular e Ionic. En este proyecto cumplí tareas de fullstack para realizar las diferentes integraciones con los diferentes proveedores de ecommerce por medio de diferentes microservicios.
